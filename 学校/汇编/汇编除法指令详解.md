# 汇编除法指令详解

除法指令：**div**

语法：div reg

```assembly
mov eax, 40h
mov bl, 2
div bl
;eax = 20h
```

### 解释：

**div**指令隐含使用通用寄存器作为被除数，关系如下表所示：

| 被除数  | 除数      | 商   | 余数 |
| ------- | --------- | ---- | ---- |
| AX      | reg/mem8  | AL   | AH   |
| DX:AX   | reg/mem16 | AX   | DX   |
| EDX:EAX | reg/mem32 | EAX  | EDX  |
